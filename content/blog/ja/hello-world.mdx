## はじめに

このブログを開設しました！Next.js 15 (App Router) + MDX + Tailwind CSS で構築し、GitHub Pages でホスティングしています。

## 技術スタック

このブログは以下の技術で構築されています：

- **Next.js 15** — App Router を使用した静的サイト生成
- **MDX** — Markdown + JSX でブログ記事を記述
- **Tailwind CSS** — ユーティリティファーストの CSS フレームワーク
- **TypeScript** — 型安全な開発
- **GitHub Pages** — 無料の静的サイトホスティング

## こだわりポイント

### 日英バイリンガル対応

日本語と英語の両方でコンテンツを提供しています。ヘッダーの言語切替ボタンから簡単にい切り替えられます。

### ダークモード

ライトモードとダークモードを切り替えられます。

### シンプルなデザイン

読みやすさを重視したミニマルなデザインにしました。
